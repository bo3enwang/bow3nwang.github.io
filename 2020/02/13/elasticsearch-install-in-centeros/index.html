CentOS 单节点部署 ElasticSearch

2020-02-13
Linux
运维

环境

  • 系统：CENTOS 7；
  • Elasticsearch 版本：elasticsearch-7.6.0-linux-x86_64

安装步骤

1.下载

在官方网站

https://www.elastic.co/cn/downloads/elasticsearch

选择适合你的版本下载

2.解压

tar xzvf elasticsearch-7.6.0-linux-x86_64.tar.gz

3.运行

进入到 Elasticsearch 的解压目录

注意事项

1. Elasticsearch 无法在 root 用下运行，在运行时候注意切换用户

创建用户方案

liunx 创建新用户

adduser newUser

然后给创建的用户加密码

passwd newUser

输入上面的命令后会要求输入两次密码。

切换刚才创建的用户su newUser 然后执行 elasticsearch 会显示 Permission denied 权限不足。

切换到 root 用户,给新建的 newUser 赋权限

chown -R XXX /elasticsearch ( elasticsearch 安装目录。)

2. Elasticsearch 默认会占用 9200 和 9300 端口

3. Elasticsearch 默认无法外网访问，如需外网访问需要进行配置

4. 运行时如果出现 max virtual memory areas vm.max_map_count [65530] is too low, increase to at least [262144] 错误

解决方案

切换到 root 用户修改配置 sysctl.conf

vi /etc/sysctl.conf

添加下面配置：

vm.max_map_count=655360

执行命令

sysctl -p
